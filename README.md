# Message Character Counter

This is a simple web project built using HTML, CSS, and JavaScript.
It allows users to type a message in a textarea and shows how many characters have been used out of 200 characters.

The character count updates automatically while the user is typing.

# 🚀 Features

Shows the number of characters typed
Maximum limit of 200 characters
Displays a message when the limit is reached
Simple and clean design
Responsive layout for mobile screens

# 🛠 Technologies Used

HTML – Structure of the webpage
CSS – Styling and layout
JavaScript – Character counting functionality
